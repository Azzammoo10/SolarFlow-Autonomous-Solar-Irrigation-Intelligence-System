# Components List — Greenhouse IoT

List of hardware components used in the SolarFlow Greenhouse Intelligence System.

| Component | Description / Type |
|-----------|--------------------|
| **Arduino Mega R3** | Main MCU |
| **Solar Panel** | Power Source |
| **Battery** | Energy Storage |
| **DHT11** | Temperature & Humidity Sensor |
| **Soil Moisture Sensor** | Analog Sensor |
| **CO2 Sensor** | Analog Sensor (A1) |
| **LDR Module** | Light Sensor (A2) |
| **Rain Sensor Module** | Digital Sensor (D3) |
| **Flame Sensor** | Digital Sensor (D16) |
| **Tank-B Level Sensor** | Reservoir Monitoring (A3) |
| **Motor / PWM Servo** | Actuators (Window/Vent) |
| **Relay** | Pump/Valve/Heating Control |
| **Pump / Valve** | Irrigation Actuators |
| **Buzzer / Alarm LED** | Alert System |
| **PCF8574** | I2C I/O Expander |
| **LM041L** | LCD Display (I2C) |
| **1N4148** | Signal Diode |
| **2N2222** | NPN Transistor |
| **Resistors / Potentiometer** | Passive Components |
| **Buttons / Switches** | Manual Controls |
